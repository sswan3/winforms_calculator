# WinForms Calculator

## Overview
A simple Windows Forms application developed in C# to practice event-driven programming,
object-oriented design, and user input validation.

## Features
- GUI-based calculator supporting basic arithmetic operations
- Separation of UI and calculation logic
- Input validation and error handling
- Clean, modular class structure

## Technologies
- C#
- Windows Forms
- .NET

## Purpose
This project was created as part of coursework to gain hands-on experience with
desktop application development and object-oriented programming principles. Run the main class in your IDE or terminal
